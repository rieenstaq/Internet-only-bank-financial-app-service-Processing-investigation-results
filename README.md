# 📊 인터넷전문은행 금융앱 서비스 만족도 조사 보고서

## 🧾 프로젝트 개요

인터넷전문은행 금융앱 사용자 경험을 분석하고, 만족도 기반 개선 방향을 도출하기 위한 설문조사 프로젝트입니다.
비대면 중심 금융 서비스 환경에서 **앱 품질이 곧 경쟁력**이라는 점에 초점을 맞추었습니다.

---

## 🎯 조사 목적

### ✔ 필요성

* 인터넷전문은행은 **무점포 운영** → 고객과의 직접 접점 부족
* 금융앱이 **핵심 커뮤니케이션 채널**
* 타 금융앱(시중은행 포함) 대비 **경쟁력 확보 필요**

### ✔ 목적

1. 금융앱 서비스 만족도 수준 진단
2. 사용자 니즈 기반 개선 포인트 도출
3. 서비스 품질 향상 방향 제시

### ✔ 기대 효과

* 차별화된 금융앱 경쟁력 확보
* 고객 만족도 향상
* 신규 고객 유입 증대

* 최고 중요: **속도 (70점)**
* 최저 중요: **이벤트 (49점)**

---

### 3️⃣ 사용자 특성별 인사이트

#### 👩 성별

* 여성: 전반적으로 높은 만족도
* 남성: 전반적으로 낮은 만족도

#### 🎂 연령

* 20대 → 서비스 품질 불만족 (특히 오류율)
* 40~50대 → 서비스 만족도 높음

#### 📱 사용량

* 사용 빈도 ↑ → 만족도 ↑
* 주 1회 사용자 → 전반적으로 낮은 만족도
